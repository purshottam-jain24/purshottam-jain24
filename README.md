### About

I'm a full-stack developer based in Delhi. I like building things that actually ship and get used, not just demos. Most of my work lives in the Next.js and React world on the front end, with backends in Node.js, Go, and Fastify.

Lately I've spent a lot of time on multi-tenant SaaS, real-time systems, and AI features that do real work. Between a couple of my projects, around 10,000 people use what I've built. I usually own things start to finish — writing the code, wiring up the infra, and keeping it running on AWS, GCP, or a plain VPS.

<br>

### What I do

- Build full products end to end, from the first empty file to something people log into.
- Design backends and APIs that hold up as traffic grows — REST and GraphQL, auth, payments, real-time data.
- Add AI features that earn their keep: voice interviews, transcription, generated content. Built on Gemini, run in production.
- Handle the boring-but-important parts too — Docker, CI/CD, self-hosting, caching, and chasing down slow queries.
